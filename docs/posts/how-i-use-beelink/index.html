How I use Beelink

I successfully installed Debian 13 (Trixie) onto a Beelink SEi8. There are a couple of important points to note:

  • To access the boot menu, unsure the USB keyboard is plugged directly in to the Beelink, not via a hub.

  • Install firmware-intel-graphics in order to get the full graphics resolution in X.
