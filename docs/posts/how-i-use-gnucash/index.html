How I use Gnucash

I use Gnucash. It does the job but often when I upgrade I encounter a new bug. These are the ones I’ve encountered, and their fixes:

  • delete and backspace don’t work in gnucash

  • gnucash 3.3 crashes upon keypress in transaction

(Interestingly both bugs related to GTK)

  • new tabs open to right of current tab, not to the right of rightmost tab

    Actually this is a new behaviour, not a bug, but I find it annoying.

Dark mode

See the GTK settings.ini configuration and then also do Edit -> Preferences -> Settings -> Register -> Disable “Use Gnucash built-in color theme”
